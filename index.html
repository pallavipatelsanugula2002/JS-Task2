var keyword

The var statement declares a variable. Variables are containers for storing information. Creating a variable in JavaScript is called "declaring" a variable The var statement declares function-scoped or globally-scoped variables, optionally initializing each to a value. using var keyword declaration,assignment,initialization,redeclaration,reassignment,reinitialization everything is possible

let keyword

let has local scope,it has introduced in the version of ES6.let values can't be redeclared,reinitialized but they can be reassigned. must be declared before their use because of the concept called hoisting. variables with let are also hosted to the top of the block but not initialized. the variables with let keyword can be redeclared outside the scope because they have local scope itself

const keyword

const has local scope,it has introduced in the version of ES6.let values can't be redeclared,reinitialized and reassigned.const values can't be directly declared or assigned,they can be only initialized.
